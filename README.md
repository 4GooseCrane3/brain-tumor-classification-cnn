# Brain Tumor Detection Using Deep Learning (CNN)

## Overview
A deep learning project to classify brain MRI images into four categories:
Glioma, Meningioma, Pituitary Tumor, and No Tumor.

## Dataset
- Source: Kaggle Brain MRI Dataset
- Classes: 4
- Images: ~7000 MRI scans

## Model
- Architecture: EfficientNetB0 + CNN classifier
- Framework: TensorFlow / Keras
- Transfer Learning + Fine-Tuning

## Results
- Training Accuracy: 97.3%
- Test Accuracy: 98.0%

## Features
- Data augmentation
- Transfer learning
- Fine-tuning for high accuracy
- Confusion matrix & classification report

## Tools
Python, TensorFlow, Keras, NumPy, Matplotlib
